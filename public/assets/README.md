# Asset Directory for Coachstate.online

Place your image files in this folder (`public/assets/`):

1. **Logo File**: `coachstate-logo.png` (or `coachstate-logo.svg`)
2. **ANLP Accreditation Logo**: `anlp-accredited-logo.png`
3. **Lukmaan Headshot / Profile Photo**: `lukmaan-profile.jpg` (or `lukmaan-profile.png`)

The site will automatically display your custom image files when present!
